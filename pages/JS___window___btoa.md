category:: Programming
type:: #Javascript
alias:: window.btoa

- > `a` 是代表 ASCII ，`b` 是代表 binary
  將 ASCII 的資料轉成 binary
  將 base64 編碼過的字串轉回原始的形式
- ## Syntax
	- ```js
	  btoa(stringToEncode);
	  ```
	- **Parameters**
		- **`stringToEncode`**
			- Type: `string`
	- **Return**
		- Type: `string`
		-
