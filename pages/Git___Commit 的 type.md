type:: #Git, #Note
alias:: Git commit 的 type

- **feat**: (Feature) 新增/修改功能
- **fix**: (Bug fix) 修正 Bug
- **docs**: (Documentation) 新增/修改文件
- **style**: 修改程式碼格式或風格，不影響原有功能
- **refactor**: 重構 or 優化，非 Bug, 新增功能等等
- **test**: 增加測試功能
- **chroe**: 增加或修改第三方套件
