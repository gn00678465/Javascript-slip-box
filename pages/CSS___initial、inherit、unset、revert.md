category:: Programing
type:: #CSS
alias:: initial、inherit、unset、revert

- ### `initial`
	- > 恢復為該 css 屬性的初始值
- ### `inherit`
	- > 繼承最近的父層 css 屬性值
- ### `unset`
	- > 有繼承性時等同 `inherit`
	  無繼承性時等同 `initial`
- ### `revert`
	- > 先恢復為瀏覽器預設；
	  無瀏覽器預設則等同 `unset`
